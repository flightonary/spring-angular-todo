# Sample Project
Spring Boot 2.4 + Angular 11 + Gradle

with;
- Spring Web
- Spring Security

# Virtual Env
bash, zsh
```
. venv/activate
```

fish
```
. venv/activate.fish
```

# Ng
```
npx ng serve 
```

# Build jar
```
gradlew bootJar
```

# Run jar
```
java -jar ./api/build/libs/api-0.0.1-SNAPSHOT.jar &
open http://localhost:8080
```
